# opencv java bindings

Add this to your pom.xml

```xml
<repositories>
    <repository>
        <id>opencv</id>
        <url>https://rawgit.com/sshaddicts/opencv/repository</url>
    </repository>
</repositories>
```

```xml
<dependency>
    <groupId>sshaddicts.opencv</groupId>
    <artifactId>opencv</artifactId>
    <version>${version}</version>
</dependency>
```
